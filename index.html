The first process that should be taken care of during the Data Mining process is the Preprocessing of Data. The Data can be available to us in any format, it may be big, so that we are not able to process all the data at once, or it may have useless attributes which are not required to be processed and will only take disc space while running an algorithm on them. We may even be acquiring data from different sources and hence might need to organise them such that they are compatible with each other. It is possible that the data has some missing values, noisy data etc. which need to be taken care of first.
In order to convert that data according to the needs of the algorithms being applied on them, we need to apply preprocessing. Data Preprocessing includes cleaning, normalisation, transformation, feature extraction, selection, reduction etc. The product of data preprocessing is the final training set.
